# InnerGrowth Skills

A collection of translation and language skills for agent tools.

## Skills

| Skill | Description |
|---|---|
| [Corporate Translator](./skills/corporate-translator/) | Translates casual workplace messages into professional corporate English |
| [Data Populator](./skills/data-populator/) | Converts natural language vocabulary into structured JSON format |
| [Grammar Coach](./skills/grammar-coach/) | Evaluates and refines English sentences to native-level fluency |
| [Sentence By Sentence Translator](./skills/sentence-by-sentence-translator/) | Provides sentence-by-sentence translation between any language pair |
| [Tri-Tone Translator](./skills/tri-tone-translator/) | Generates three parallel translation variants across different registers |

## Usage

```bash
pnpm dlx skills add github:jeremy-ww/skills
```

To update:
```bash
pnpm dlx skills update
```
